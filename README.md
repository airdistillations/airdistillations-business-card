# A·I·R Distillations — Founder Cards

Single-page digital business card for the three founders of A·I·R Distillations Antwerp. Mobile-first, static HTML/CSS/JS — no build step.

## Run locally

Open `index.html` in any browser, or serve the folder:

```bash
python3 -m http.server 8000
# → http://localhost:8000
```

## Deploy

Drop the whole folder onto any static host (Netlify, Vercel, Cloudflare Pages, GitHub Pages, S3, etc.). No build needed.

## Structure

- `index.html` — full page (CSS + JS inline)
- `assets/` — logos and team photo
